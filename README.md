## 👨‍💻 About Me
I am a passionate Full-Stack Developer and Security Researcher with a profound interest in **AI** and **CyberSecurity**. I specialize in building highly secure, dynamic, and visually stunning web applications.

This repository contains the complete source code for my personal portfolio website, designed to showcase my technical skills, professional experience, certifications, and live coding statistics.

## ✨ Key Features
- **Immersive 3D Background**: Features an interactive, hardware-accelerated sky blue particle system and floating meshes built with `Three.js` and `React Three Fiber`.
- **Live Coding Statistics**: Dynamically fetches and aggregates my real-time problem-solving counts directly from my **LeetCode** and **CodeChef** profiles using custom `@tanstack/react-query` hooks and proxy APIs.
- **Glassmorphism Design System**: A completely custom, premium UI aesthetic featuring layered blurs, subtle borders, and smooth micro-animations.
- **Dynamic Content Loading**: Automated galleries that instantly pull and format my certifications and offer letters from local asset directories.
- **Responsive Architecture**: Fully mobile-optimized layout built with modern CSS flexbox and grid paradigms.

## 🛠️ Technology Stack
This portfolio was architected and built using the following modern web technologies:

- **Framework**: [React 18](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **3D Graphics**: [Three.js](https://threejs.org/) & [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/)
- **State & Data Fetching**: [React Query (TanStack)](https://tanstack.com/query/latest)
- **Icons**: [Lucide React](https://lucide.dev/)
- **AI Tooling**: Img.ly Background Removal Node (for programmatic avatar adjustments)

## 🏗️ How It Was Made
The website was bootstrapped strictly as a local-first single-page application (SPA). 
1. The foundational structure was laid out using React and Vite for blisteringly fast HMR.
2. The UI was iteratively designed using Tailwind CSS utility classes, heavily reliant on custom CSS variables (`index.css`) to enforce the signature "Sky Blue" developer aesthetic.
3. Complex framer-motion variants were injected into components like `HeroSection` and `ExperienceSection` to ensure content beautifully slides and fades into view.
4. Finally, API hooks were integrated into a global `QueryClientProvider` to pull in live competitive programming statistics.
